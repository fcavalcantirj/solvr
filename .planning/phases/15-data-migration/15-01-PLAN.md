    - ListQuorumRooms: `SELECT id, slug, display_name, token_hash, owner_id, created_at FROM rooms ORDER BY created_at`
    - ListQuorumMessages: `SELECT agent_name, content, created_at FROM messages WHERE room_id = $1 ORDER BY created_at`
    - FindSolvrUserByEmail: `SELECT id FROM users WHERE email = $1`
    - CheckSlugConflict: `SELECT EXISTS(SELECT 1 FROM rooms WHERE slug = $1 AND id != $2)`

Compute sequence_num in Go by iterating (index+1) since messages are already ordered by created_at from the Quorum query. Each INSERT: `INSERT INTO messages (room_id, author_type, author_id, agent_name, content, content_type, metadata, sequence_num, created_at) VALUES ($1, $2, $3, $4, $5, $6, $7::jsonb, $8, $9)`. Return count of inserted messages.
    - UpdateRoomMessageCount: `UPDATE rooms SET message_count = (SELECT COUNT(*) FROM messages WHERE room_id = $1 AND deleted_at IS NULL) WHERE id = $1` (D-06)
    - BeginTx: delegates to solvr pool's BeginTx
