
## Task 2: /referrals Dashboard Page

**Requirements:** DASH-01, DASH-02, DASH-03, DASH-04, REF-05

**Files to create:**
- `frontend/app/referrals/page.tsx`
- `frontend/app/referrals/page.test.tsx`

### Page Behavior

1. `'use client'` — needs clipboard API, useState, useEffect
2. On mount: check `isAuthenticated` from `useAuth()`:
   - If `isLoading`, show skeleton
   - If not authenticated, redirect to `/login?next=/referrals` via `router.push()`
   - If authenticated, fetch `api.getMyReferral()`
3. Display referral code with copy button
4. Display referral count
5. Display share links: X/Twitter pre-filled tweet + copyable referral URL

### Layout Pattern

Follow existing page patterns (see `frontend/app/leaderboard/page.tsx`):
- `<Header />` at top
- `<main className="pt-20">` content area
- `max-w-7xl mx-auto px-4 sm:px-6` container
- Font: `font-mono` throughout (project convention)
- Styling: Tailwind v4, `bg-background`, `text-foreground`, `border-border` tokens

### Copy Button

```typescript
const [copied, setCopied] = useState(false);

const handleCopy = async (text: string) => {
  await navigator.clipboard.writeText(text);
  setCopied(true);
  setTimeout(() => setCopied(false), 2000);
};
```

Show "Copied!" text or check icon for 2 seconds, then revert.

### Pre-filled Tweet Link (satisfies PAGE-03)

```typescript
const referralUrl = `https://solvr.dev/join?ref=${referralCode}`;
const tweetText = "I'm using @SolvrDev to solve programming problems faster. Join me:";
const tweetLink = `https://twitter.com/intent/tweet?text=${encodeURIComponent(tweetText)}&url=${encodeURIComponent(referralUrl)}`;
```

### Dashboard Sections

1. **Referral Code Card** — code displayed in monospace, copy button beside it
2. **Stats Card** — referral count (e.g., "3 successful referrals")
3. **Share Section** — two actions:
   - "Share on X" button (opens tweet intent in new tab)
   - "Copy referral link" button (copies `https://solvr.dev/join?ref=CODE`)

### States

- **Loading:** skeleton loader (animate-pulse divs) while `isLoading` or API fetch in progress
- **Error:** "Failed to load referral data" message with retry
- **Success:** full dashboard
