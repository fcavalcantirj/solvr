# Plan 02-01: Database Migration + Email Broadcast Repository

## Objective

The requirement text is descriptive, not prescriptive about table names.

    subject TEXT NOT NULL,
    body_html TEXT NOT NULL,

```go
package db

import (
	"context"
	"fmt"
	"time"

	"github.com/fcavalcantirj/solvr/internal/models"
)

// EmailBroadcastRepository handles database operations for email broadcast logs.
type EmailBroadcastRepository struct {
	pool *Pool
}

// NewEmailBroadcastRepository creates a new EmailBroadcastRepository.
func NewEmailBroadcastRepository(pool *Pool) *EmailBroadcastRepository {
	return &EmailBroadcastRepository{pool: pool}
}
```

Three methods:

1. **`CreateLog(ctx context.Context, broadcast *models.EmailBroadcast) (*models.EmailBroadcast, error)`**
   - INSERT with `subject`, `body_html`, `body_text`, `total_recipients`, `status`
   - Use `RETURNING id, subject, body_html, body_text, total_recipients, sent_count, failed_count, status, started_at, completed_at, created_at`
   - Scan all 11 fields (completed_at into `*time.Time`)
   - On error: `fmt.Errorf("create email broadcast log: %w", err)`
   - Return pointer to scanned struct

2. **`UpdateStatusAndCounts(ctx context.Context, id string, status string, sentCount, failedCount int, completedAt *time.Time) error`**
   - UPDATE `email_broadcast_logs SET status = $2, sent_count = $3, failed_count = $4, completed_at = $5 WHERE id = $1`
   - Use `pool.Exec`
   - Check `RowsAffected() == 0` → return `ErrNotFound`
   - On error: `fmt.Errorf("update email broadcast log: %w", err)`

3. **`List(ctx context.Context) ([]models.EmailBroadcast, error)`**
   - SELECT all columns `FROM email_broadcast_logs ORDER BY started_at DESC`
   - Scan rows in loop (same 11 fields)
   - If result is nil, return `[]models.EmailBroadcast{}` (empty slice, not nil)
   - On error: `fmt.Errorf("list email broadcast logs: %w", err)`

Note: `body_text` is nullable in the DB. Use `*string` for scanning in `CreateLog` and `List`, then dereference into `EmailBroadcast.BodyText` (empty string if NULL). Alternatively, use `COALESCE(body_text, '')` in the SQL.
