## Objective

Add per-recipient template variable substitution to the existing `BroadcastEmail` admin handler. Three variables: `{name}`, `{referral_code}`, `{referral_link}`. Applied to both `body_html` and `body_text` before each email send. Dry-run shows substituted preview for the first recipient.

#### 3b: GREEN — Implement the helper

```go
// substituteTemplateVars replaces {name}, {referral_code}, and {referral_link}
// in the given body string with the provided per-recipient values.
func substituteTemplateVars(body, name, code, link string) string {
    r := strings.NewReplacer("{name}", name, "{referral_code}", code, "{referral_link}", link)
    return r.Replace(body)
}
```

**Verify GREEN:** `go test ./internal/api/handlers/ -run TestSubstituteTemplateVars` — all subtests pass.

### Step 4: Integrate substitution into BroadcastEmail send loop

**File:** `backend/internal/api/handlers/admin.go` (lines 184-193)

In the send loop, before calling `h.emailSender.Send`, compute the substituted body for each recipient:

```go
for _, recipient := range recipients {
    referralLink := ""
    if recipient.ReferralCode != "" {
        referralLink = "https://solvr.dev/join?ref=" + recipient.ReferralCode
    }
    personalHTML := substituteTemplateVars(req.BodyHTML, recipient.DisplayName, recipient.ReferralCode, referralLink)
    personalText := substituteTemplateVars(req.BodyText, recipient.DisplayName, recipient.ReferralCode, referralLink)

    err := h.emailSender.Send(ctx, recipient.Email, req.Subject, personalHTML, personalText, unsubHeaders)
    ...
}
```

**Key detail:** Build `referralLink` from the code. If code is empty, link is empty too (no broken URL).

### Step 6: Update dry-run to show substituted preview

**File:** `backend/internal/api/handlers/admin.go` (lines 146-153)

Change the dry-run response to include a `preview` object showing the substituted body for the first recipient:

```go
if req.DryRun {
    preview := map[string]string{}
    if len(recipients) > 0 {
        first := recipients[0]
        referralLink := ""
        if first.ReferralCode != "" {
            referralLink = "https://solvr.dev/join?ref=" + first.ReferralCode
        }
        preview["body_html"] = substituteTemplateVars(req.BodyHTML, first.DisplayName, first.ReferralCode, referralLink)
        preview["body_text"] = substituteTemplateVars(req.BodyText, first.DisplayName, first.ReferralCode, referralLink)
    }
    writeAdminJSON(w, http.StatusOK, map[string]interface{}{
        "would_send": len(recipients),
        "recipients": recipients,
        "preview":    preview,
    })
    return
}
```

5. Assert `preview` does NOT contain raw `{name}` or `{referral_code}` or `{referral_link}` tokens.
